IAA LTV 计算口径（生命周期累积式）

⚠️ LTV 为累积口径：LT7 包含 Day0~Day6 的所有广告收入，LT30 包含 Day0~Day29，以此类推。不要用分段口径。
生命周期标签life_day 范围公式
LT7life_day BETWEEN 0 AND 6SUM(advalue) / COUNT(DISTINCT new_user)
LT30life_day BETWEEN 0 AND 29同上，窗口扩大
LT60life_day BETWEEN 0 AND 59同上
LT90life_day BETWEEN 0 AND 89同上
LT120life_day BETWEEN 0 AND 119同上
LT180life_day BETWEEN 0 AND 179同上

其中 life_day = DATE_DIFF('day', DATE(reg_date), DATE(active_date))；新用户定义：been_reg_days = 1

IAA LTV + eCPM + Freq 分析 SQL 结构（通用模板）

-- 通用 CTE 结构（适用 UNO/P10/SKB/UNO2）:
-- 1. new_users: 锁定注册窗口内新用户 (been_reg_days=1)
-- 2. daily_ad_metrics: JOIN 活跃表，计算 life_day、每日 RV/INT 次数和收入
-- 3. lifecycle_stage: VALUES 表定义 LT 阶段 (LT7~LT180)
-- 4. user_lt_metrics: 按用户×平台×LT阶段聚合
-- 5. 最终 SELECT: GROUP BY CUBE(platform) 输出分平台 + ALL

-- 核心计算：
-- Freq/DAU     = SUM(adplay_cnt) / SUM(active_days)
-- eCPM         = SUM(ad_revenue) * 1000 / NULLIF(SUM(adplay_cnt), 0)
-- IAA LTV      = SUM(ad_revenue) / COUNT(DISTINCT user_id)

-- 剔除 Banner 口径:
-- TOTAL 收入 = advalue_reward_sum_1d + advalue_interstitial_sum_1d  (非 advalue_sum_1d)
-- TOTAL 次数 = rv_play_cnt + int_play_cnt

UNO2 广告数据特殊处理（FULL OUTER JOIN）

-- UNO2 广告数据不在活跃表，需从明细日志表分别取播放次数和收入再合并
-- 播放次数: dw_ods_mn08.c_client_app_ad_log WHERE log_subtype='adplay' AND status=1
-- 广告收入: dw_ods_mn08.c_client_app_ad_log WHERE log_subtype='advalue'
-- 安卓收入需 /1000000:
--   CASE WHEN UPPER(platform)='IOS' THEN CAST(value AS DOUBLE)
--        ELSE CAST(value AS DOUBLE)/1000000 END
-- 合并方式: FULL OUTER JOIN ON date + role_id + platform
-- 最终字段: COALESCE(play.rv_cnt, 0), COALESCE(value.rv_revenue, 0) ...

1. 广告 log_type 映射标准

-- ads_custom_logtype_player_date / 原始日志中通用的 log_type 映射
CASE
  WHEN log_type = 'adinit' AND status = 1 THEN 'init'       -- 广告初始化
  WHEN log_type = 'adentrance'            THEN 'entrance'    -- 进入广告位
  WHEN log_type = 'advisit' AND action = 'play' THEN 'click' -- 点击播放
  WHEN log_type = 'adplay' AND status = 1 THEN 'play'        -- 播放成功
  WHEN log_type = 'adplay' AND status <> 1 THEN 'playFail'  -- 播放失败
  WHEN log_type = 'adload' AND status = 1 THEN 'load'        -- 加载成功
  WHEN log_type = 'adload' AND status <> 1 THEN 'loadFail'  -- 加载失败
  WHEN log_type = 'adclosed'              THEN 'closed'      -- 关闭
  WHEN log_type = 'adrewarded'            THEN 'rewarded'    -- 奖励发放
  WHEN log_type = 'advalue'               THEN 'advalue'     -- 收益回调
END

  select date, platform, account_id, 'play', null

  from dw_ods_mn04.c_advalue
  where date between '2025-01-19' and '2025-02-19'

  count(distinct if(log_type='login', role_id, null)) as DAU,

  sum(if(log_type='advalue', revenue, 0)) as Revenue
